	Task : " -[ ] " 를 사용하는 체크 박스 형태의 머리글. 일시적이며 추적관리 필요
		할 일, 상태
>  파일 분류법
	Link : " [[]] " 를 사용해 생성. 노트나 그룹을 연결 가능
		- 링크 자체가 파일
		- 백링크, 아웃링크로 생각의 발상 방향 표시 가능
		- 메타 데이터로 파일 간 관계 정의
		- Aliases(별칭) 사용 가능
	Task : " # " 를 사용해 생성. 키워드로 분류
		- 파일이 아니고 의미만 가진다.
		- 계층구조 사용가능
		- 플러그인으로 일괄 변경 가능
		- 검색용의
		- Dataview, Tasks 쿼리문 작성 편함
